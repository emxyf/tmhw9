# tmhw9
This repository contains the R code and data file I used for my dissertation.

"full_code" contains all segments of the code I used, pieced together into one document. 

"initial_cleaning" contains the first steps I took to clean the data and set it up for further analysis.

"descriptive_statistics_robustness" contains the code used to calculate the descriptive statistics and to carry out multicollinearity checks.

"regressions" contains the code used to create all the survey-weighted logistic regressions for both stages.

"bsa2024.sav" contains the BSA 2024 survey data I based my analysis off of.
